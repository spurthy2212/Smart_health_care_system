Cloud-Based Secure and Efficient Smart Healthcare System using IoMT and Cloud Computing with Public Verifiability
📌 Project Overview

This project is a Smart Healthcare Monitoring System designed using IoMT (Internet of Medical Things) and Cloud Computing concepts. It helps in monitoring patient health data and securely storing the information in the cloud while ensuring data integrity through public verifiability.

The system provides a web-based interface where healthcare providers can access patient data, monitor health parameters, and verify data securely.

🎯 Objectives

To develop a smart healthcare monitoring platform.

To integrate IoMT-based health data with cloud storage.

To ensure secure data management and verification.

To enable remote monitoring of patient health records.

✨ Features

Patient health data monitoring

Secure cloud data storage

Public verifiability for data integrity

User-friendly web interface

Real-time health data visualization
